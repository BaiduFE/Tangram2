baidu.array().hash() 示例

将两个数组参数合并成一个类似hashMap结构的对象，这个对象使用第一个数组做为key，使用第二个数组做为值，如果第二个参数未指定，则把对象的所有值置为true

  • 语法: baidu.array().hash(values)
  • 参数1: {Object} values 匹配项
  • 返回: {Array} 映射后的HASH对象

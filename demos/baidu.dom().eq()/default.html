baidu.dom().eq()

根据索引获得Tangram Dom集合总指定的元素

调用说明

  • 语法: baidu.dom(args).eq(index)
  • 参数: index指定索引，如果为负数，则以最后一个元素为-1开始索引
  • 返回: 仅包含指定元素的TangramDom对象
高亮指定索引的li
  • baidu.dom().eq()
  • 可以选择集合中具有指定索引值的元素
  • 如果索引为非负，则第一个元素为 0 开始索引
  • 如果索引为负数，则以最后一个元素为 -1 开始索引
  • 由于使用数字索引
  • 此方法可以用来做一些轮询效果
  • 比如本Demo

baidu.dom("ul.list li").removeClass("hl").eq().addClass("hl");

baidu.object.merge() 示例

合并源对象的属性到目标对象

  • 语法: baidu.object.merge(target, source[, opt_options])
  • 参数1: {Function} target 目标对象
  • 参数2: {Function} source 源对象
  • 参数3: {Object} opt_options optional merge选项
  • 参数4: {boolean} overwrite optional 如果为真，源对象属性会覆盖掉目标对象上的已有属性，默认为假.
  • 参数5: {string[]} whiteList optional 白名单，默认为空，如果存在，只有在这里的属性才会被处理
  • 参数6: {boolean} recursive optional 是否递归合并对象里面的object，默认为否
  • 返回: {Object} merge后的object

baidu.cookie.setRaw() 示例

设置cookie的值，不对值进行编码

  • 语法: baidu.cookie.setRaw(key, value[, options])
  • 参数: {string} key 需要设置Cookie的键名
  • 参数: {string} value 需要设置Cookie的值
  • 参数: {Object} options 设置Cookie的其他可选参数
  • 参数: {string} path cookie路径
  • 参数: {Date|number} expires cookie过期时间,如果类型是数字的话, 单位是毫秒
  • 参数: {string} domain cookie域名
  • 参数: {string} secure cookie是否安全传输
  • 返回: undefined
